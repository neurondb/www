### Template 1: Directory Submission
```
Subject: [Submission] NeuronDB - PostgreSQL AI Extension

Hi [Name],

I'd like to submit NeuronDB for inclusion in [Directory Name].

NeuronDB is an open-source PostgreSQL extension that adds vector search, 
ML inference, and RAG pipeline capabilities directly in PostgreSQL. It's 
designed as a production-ready alternative to standalone vector databases.

Key features:
- HNSW, IVFFlat, and DiskANN indexing
- 52+ ML algorithms in pure C
- GPU acceleration (CUDA, ROCm, Metal)
- 473 SQL functions
- Complete RAG pipeline

Links:
- Website: https://neurondb.ai
- GitHub: https://github.com/neurondb-ai/neurondb
- Documentation: https://neurondb.ai/docs

Would you consider adding NeuronDB to your list?

Thank you,
[Your Name]
```
